# 图片资源文件夹

此文件夹用于存放网站所需的图片资源。

## 📸 添加个人照片

### 快速步骤

1. 准备一张正方形的个人照片
2. 使用工具压缩图片（推荐 TinyPNG）
3. 将照片命名为 `profile.jpg`
4. 放入此文件夹
5. 网站会自动显示

### 照片规格建议

- **文件名**：`profile.jpg` 或 `profile.png`
- **尺寸**：400x400px 或更大（正方形）
- **格式**：JPG 或 PNG
- **大小**：建议 < 200KB
- **显示**：以 160x160px 圆角矩形显示

## 🖼️ 其他图片

### Favicon
- 文件名：`favicon.png`
- 尺寸：32x32px 或 64x64px
- 格式：PNG

### Apple Touch Icon
- 文件名：`apple-touch-icon.png`
- 尺寸：180x180px
- 格式：PNG

## 🛠️ 图片优化工具

推荐使用以下工具压缩图片：

- [TinyPNG](https://tinypng.com/) - 在线压缩
- [ImageOptim](https://imageoptim.com/) - macOS 应用
- [Squoosh](https://squoosh.app/) - Google 在线工具

## 📱 响应式说明

照片在不同设备上的显示：
- **桌面**：160x160px，左侧显示
- **平板**：120x120px，左侧显示
- **手机**：140x140px，居中显示

## 💡 提示

- 如果没有照片，主页会自动隐藏照片区域
- 只上传愿意公开的照片
- 定期清理不需要的图片文件
- 建议使用高质量、光线充足的照片
