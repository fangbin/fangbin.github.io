欢迎来到我的个人博客！这是第一篇博文，介绍一下这个博客的功能和特点。

## 博客功能

这个博客系统具有以下功能：

### 自动文章生成

只需要在 `_posts` 文件夹中添加 Markdown 文件，Jekyll 会自动将其转换为网页。文件命名格式为：

```
YYYY-MM-DD-title.md
```

例如：`2024-01-15-welcome.md`

### 分类和标签

每篇文章都可以设置分类和标签：

```yaml
---
layout: post
title: "文章标题"
date: 2024-01-15
categories: [技术]
tags: [Jekyll, 博客, 教程]
---
```

### Markdown 支持

支持完整的 Markdown 语法，包括：

- 标题（H1-H6）
- 列表（有序和无序）
- 代码块
- 引用
- 链接和图片
- 表格

#### 代码示例

```python
def hello_world():
    print("Hello, World!")
    return True

if __name__ == "__main__":
    hello_world()
```

## 开始写作

要创建新文章，只需：

1. 在 `_posts` 文件夹创建新的 `.md` 文件
2. 添加 Front Matter（YAML 头部信息）
3. 编写 Markdown 内容
4. 提交到 GitHub

就是这么简单！
